# yelp

To get started, make a .env file in the server folder. Fill out the following fields

```
PORT=3001
PGUSER=postgres
PGHOST=localhost
PGPASSWORD=Enter password fot postgres database
PGDATABASE=Enter database name created
PGPORT=5432
```
